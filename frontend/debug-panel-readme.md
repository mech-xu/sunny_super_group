# 调试面板使用说明

## 访问调试面板

在浏览器中访问以下URL即可进入调试面板：

```
http://localhost:5173/debug
```

## 功能说明

调试面板包含以下功能：

1. **页面导航** - 快速跳转到各个主要页面
2. **订单流程测试** - 测试关键订单流程
3. **UI组件测试** - 测试各类UI组件
4. **设备信息** - 显示当前设备信息

## 在Chrome中模拟iPhone设备

1. 打开Chrome浏览器，访问 `http://localhost:5173`
2. 按 `F12` 键打开开发者工具
3. 点击设备模拟图标（手机/平板图标）
4. 选择任意iPhone设备型号
5. 访问 `/debug` 路径使用调试面板

## 调试提示

- 在iPhone模拟模式下测试页面响应式设计
- 检查触摸目标是否足够大，易于点击
- 验证页面内容在小屏幕上是否正常显示
- 测试订单流程在移动端的可用性
