# sunnynow.net - Financial Workspace (Framework)

## ⚠️ Project Status: INCOMPLETE / FRAMEWORK STAGE
This repository contains the initial structural framework for the sunnynow.net project. It is currently used for prototyping and internal testing.

## Project Overview
A professional 'Financial Workspace' designed for fund managers and bank agents to manage community announcements, member audits, and feedback loops, bypassing the limitations of traditional chat group management.

## Tech Stack
- **Frontend**: Vue 3 + Tailwind CSS
- **Backend**: PostgREST + PostgreSQL
- **Infrastructure**: Local NAS $\rightarrow$ Cloudflare Tunnel $\rightarrow$ Public API
- **Bot**: Node.js / Wechaty (Integrated for broadcasting)

## Current Implementation
- [x] Basic User Authentication (Simulated)
- [x] Admin Dashboard (Publishing & Audit)
- [x] Member View (Announcement Square & Feedback)
- [x] Basic API Integration with PostgreSQL
- [x] Bot Broadcasting logic (Prototype)
