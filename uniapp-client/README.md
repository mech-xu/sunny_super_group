# SunnyNow团购自提系统 - 适用于加拿大华人群体

## 项目概述
SunnyNow团购自提系统是一个专为加拿大华人群体设计的团购平台，旨在提供便捷的商品团购和自提服务。该项目基于uni-app开发，可以同时发布到iOS、Android、H5、微信小程序等多个平台。

## 项目特点
- 专门针对加拿大市场设计，使用加元(CAD)作为货币单位
- 支持多语言（中文/英文），便于华人群体使用
- 团购模式，享受批发价格
- 自提点模式，降低配送成本
- 针对加拿大节假日和消费习惯进行优化

## 技术架构
- 前端：uni-app + Vue 3 + Pinia
- 后端：Node.js + PostgreSQL (通过PostgREST)
- 部署：Docker + Nginx
- CDN：CloudFlare

## 功能模块
1. 用户管理：注册、登录、个人信息维护
2. 商品管理：商品展示、分类、搜索
3. 购物车：商品添加、数量调整、价格计算
4. 订单管理：下单、支付、状态跟踪
5. 自提点：选择最近的自提点，获取提货码
6. 团购活动：参与团购，享受优惠价格

## 加拿大本土化特色
- 货币单位：使用加元(CAD $)
- 物流模式：自提点取货，适应加拿大地理特点
- 支付方式：集成加拿大主流支付方式
- 语言支持：中英文双语支持
- 法规合规：符合加拿大隐私保护法规

## 安装与运行
1. 安装依赖：`npm install`
2. 开发模式运行：`npm run dev:mp-weixin` 或 `npm run dev:h5`
3. 生产构建：`npm run build:mp-weixin` 或 `npm run build:h5`

## 部署说明
项目可以部署到各种平台：
- 微信小程序：上传至微信小程序平台审核发布
- H5应用：部署到Web服务器
- App应用：打包成iOS/Android应用
